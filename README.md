# ShumLehtApp

A Flutter mobile hotel booking app with a polished blue theme, login/signup flow, hotel browsing, and reservation flow.

## Run the app

1. Open the project folder in VS Code or another editor.
2. Run `flutter pub get`.
3. Start the app with `flutter run`.

## Screens

- Login screen with email/password inputs
- Sign up screen for new accounts
- Home page with search and filter chips
- Hotel detail page with amenities and pricing
- Reservation page with date selection and guest count

## Features

- Blue-themed modern UI
- Simple login and sign-up navigation
- Hotel cards with images and ratings
- Reservation confirmation flow
